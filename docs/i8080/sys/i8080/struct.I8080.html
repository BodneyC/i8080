Halts the CPU at the end of the current cycle (if using i8080::run)

Though not true to the CPU, sends an end of transmission byte to all out-devices. This allows the program to end and is akin to a shutdown signal I suppose… kinda

Adds a value to the A register

This function takes a carry argument. If false this functions acts like a [ADD A, x] instruction where as if using the real value of the carry register, this acts like [ADC A, x].

fn cmp(&mut self, val: u8)

A CMP instruction is essentially a [SUB A, x] but the result is temporary and discarded. Here, for ease, I am replacing A in self.sub(...) and then restoring it.
